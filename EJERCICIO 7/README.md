El problema consiste en desarrollar un sistema que simule el funcionamiento básico de un 
cajero automático. El programa debe permitir registrar un usuario con un saldo inicial, 
consultar el saldo disponible, realizar depósitos y efectuar retiros de dinero verificando que 
exista suficiente saldo en la cuenta. Además, la información debe almacenarse en archivos 
para conservar los datos del usuario. Para la solución se aplican conceptos de Programación 
Orientada a Objetos mediante clases y métodos, junto con el manejo de archivos para guardar 
y mostrar la información registrada.
