El problema consiste en desarrollar un sistema que permita registrar las calificaciones de un 
estudiante de manera organizada y automática. El programa debe almacenar el nombre y tres 
notas, calcular el promedio final y determinar si el estudiante aprueba o reprueba según la 
calificación obtenida. Además, se requiere aplicar Programación Orientada a Objetos, 
utilizando métodos y clases para una mejor estructura del código, junto con el manejo de 
archivos para guardar la información de forma permanente.
